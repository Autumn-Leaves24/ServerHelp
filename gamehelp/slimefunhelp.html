# 粘液科技简介

介绍

粘液科技是一个Bukkit插件，旨在一定程度上代替Mod。 使得Bukkit类服务器无需Forge即可享受Mod般的体验。 它提供了你能想到的一切，从背上背包出门探险到喷气背包遨游天际。 作者同样还加入了 API 让dalao们快速安装/编写这款插件的扩展插件。 你的服务器玩家，是不是已经厌倦了无聊的纯净生存？ 那么你绝对会喜欢这款插件！它新增了400+个物品，给服务器生存带来更多的乐趣！
